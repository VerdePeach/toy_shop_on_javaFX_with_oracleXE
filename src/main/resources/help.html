Help

Add new toy

To create new toy press "Create toy" button. You will see create toy window with fields.

"Toy name" field must be filled to create new toy. It is possible to use any characters.

"Price" field must be float number value more then 0.

"Amount" field must be integer value more then 0.

"Start age" field must be integer value more then 0. Value of this field must be less or equal then value of "End age" field.

"End age" field must be integer value more then 0. Value of this field must be more or equal then value of "Start age" field.

Check box "Clear fields after adding", select it if you want to clear field success adding. If adding will be failed, fields would noy be clear.

After filling fields press button "Add" to add or "Cancel" button to close add window.

Edit toy

To edit toy press "Edit toy" button. You will see edit toy window with fields.

"Toy id" field must be filled to change name of the toy and all other fields.

"Toy name" field must be filled to change price, amount, start age and end age of the toy.

"Price" field must be float number value more then 0.

"Amount" field must be integer value more then 0.

"Start age" field must be integer value more then 0. Value of this field must be less or equal then value of "End age" field.

"End age" field must be integer value more then 0. Value of this field must be more or equal then value of "Start age" field.

Check box "Clear fields after editing", select it if you want to clear field success adding. If editing will be failed, fields would noy be clear.

After filling fields press button "Edit" to edit or "Cancel" button to close add window.

Delete toy

To delete toy press "Delete toy" button. You will see delete toy window with fields.

You should fill one of the fields "Toy id" or "Toy name".

You can delete toys by id or name.

If you fill all fields if id and name will belong yo different toys - two toys will be deleted.

"Start age" field must be integer value more then 0. Value of this field must be less or equal then value of "End age" field.

"End age" field must be integer value more then 0. Value of this field must be more or equal then value of "Start age" field.

"Get the most expensive toys" button shows the most expensive toys.

"Start age" field must be integer value more then 0. Value of this field must be less or equal then value of "End age" field.
